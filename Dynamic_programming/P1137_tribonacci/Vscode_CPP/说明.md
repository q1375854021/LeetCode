# 说明

这个工程的用法

##### 1，source

用来存放源文件

##### 2，header

用来存放头文件

##### 3，build

用来存放编译后的文件

## 编译指令


| 命令                      | 作用                      |
| ------------------------- | ------------------------- |
| `python make.py cmake`    | 生成构建系统 (MinGW 配置) |
| `python build.py build`   | 编译 Debug 版本           |
| `python build.py release` | 编译 Release 版本         |
| `python build.py exe`     | 运行可执行文件            |
| `python build.py clean`   | 清理所有生成文件和目录    |
